# 🚀 Desafio de Código

Este repositório contém a solução de um **desafio de git e github**, com o objetivo de Praticar formatação Markdown, Commits, issues e pull requests para experimentar os processos de colaboração no Git.

---

## 📌 Descrição do Desafio

Te desafio a criar uma conta no GitHub, adicionar um repositório público com o seu README.md utilizando o GitHub.dev.
Adicione colaboradores, e pratique formatação Markdown para instruções e descrição do projeto. Realize alterações,
Commits, abra issues e pull requests para experimentar os processos de colaboração no Git.

---

## 🛠 Tecnologias Utilizadas

* Git
* GitHub
